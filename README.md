# 🚀 DeFi Dashboard

一个多链DeFi仪表板，支持钱包连接、资产跟踪、代币交换等功能。

## ✨ 功能特性
- 🔗 多链钱包连接（Ethereum、Polygon、Arbitrum）
- 📊 实时资产总览（ETH + ERC-20）
- 🔄 代币交换（Uniswap V2集成）
- 📜 交易历史查看
- ⭐ 代币收藏夹

## 🛠️ 技术栈
- **框架**: Next.js 14 + TypeScript
- **UI**: Tailwind CSS + shadcn/ui
- **区块链**: Wagmi + RainbowKit + Viem
- **状态管理**: Zustand + React Query
